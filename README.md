# シンプルTODOアプリ

HTML/CSS/JavaScriptで作成したブラウザベースのタスク管理アプリです。

## 機能

- ✅ TODOの追加・削除・編集
- ✅ 完了/未完了の切り替え
- ✅ localStorageでのデータ永続化
- ✅ フィルター機能（全て/未完了/完了済み）
- ✅ レスポンシブデザイン

## 使い方

ブラウザで `index.html` を開くだけで使えます。

```bash
open index.html
```

## 技術スタック

- HTML5
- CSS3
- Vanilla JavaScript
